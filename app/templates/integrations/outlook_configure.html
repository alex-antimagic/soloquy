Setup Instructions
1. Sign in to Azure Portal

Go to Azure Portal and sign in with your Microsoft account

2. Register App

Navigate to "Azure Active Directory" → "App registrations" → "New registration"

  • Name: "Soloquy MCP Integration"
  • Supported account types: "Accounts in any organizational directory and personal Microsoft accounts"
3. Configure Authentication

In your app's "Authentication" section:

  • Add platform: "Web"
  • Add redirect URI:
    {{ url_for('integrations.outlook_callback', scope=scope, _external=True) }}
  • Enable "Access tokens" and "ID tokens"
4. Create Client Secret

In "Certificates & secrets" → "Client secrets" → "New client secret"

  • Description: "Soloquy MCP"
  • Expiration: Your choice
  • Copy the secret value immediately (you can't see it again)
5. API Permissions

In "API permissions" → "Add a permission" → "Microsoft Graph":

  • Mail.ReadWrite (Delegated)
  • Mail.Send (Delegated)
  • offline_access (Delegated)
Security Note: Your credentials are encrypted at rest. The Outlook MCP server runs as a separate process and only your authorized agents can access it.
